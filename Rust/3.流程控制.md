# 流程控制
## if-else
- if 是表达式，有返回值，后面需要分号
```rs
let result = if condition { 
    "yes" 
} else { 
    "no" 
};
```

## match
```rs
match value {
    1 => println!("one"),
    2 | 3 => println!("two or three"),  // 多值匹配
    4..=10 => println!("four to ten"),  // 范围匹配
    _ => println!("something else"),    // 通配符
}
```

## 循环
### for-in
```rs
for i in 0..5 {           // 0 到 4
    println!("{}", i);
}
```
### while
- 每次循环调用`iterator.next()`
- 如果得到`Some(item)`，就把`item`绑定到变量，进入循环体
- 一旦得到`None`（模式匹配失败）while let 立即终止——不需要你手动 break
```rs
while let Some(item) = iterator.next() {
    // 处理 item
}
```
其中，iterator.next的定义：
```rs
fn next(&mut self) -> Option<Self::Item>;
```
