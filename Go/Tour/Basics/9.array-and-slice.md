# array

go 中 `[6]int` 表示一个长度为 6 的数组类型
- 数组的长度不可变
- 数组的长度是其类型的一部分

```go
primes := [6]int{2, 3, 5, 7, 11, 13} // 可使用大括号初始化
```

# slice

go 中 `[]int` 表示一个切片
- slice 通过指定两个索引构造，中间以冒号分隔 `arr[1:4]`
- 左闭右开（包括左边的索引，不包括右边的索引）

## slice 是引用

slice 自己并没有数据，只是引用它底层的数组，修改它的元素，会同时修改底层的数组（其他 slice 也能读取到共享部分的变化）

- [3]bool{true, true, false}
- []bool{true, true, false}
- 后者会创建前者一样的数组，然后返回一个引用它的切片

```go
// 嵌套初始化
	s := []struct {
		i int
		b bool
	}{
		{2, true},
		{3, false},
	}
	fmt.Println(s)
```

## slice 缺省值

对于 `var a [10]int`来说，以下四个切片是一样的
```go
a[0:10]
a[:10] // 从头开始
a[0:]  // 到结尾结束
a[:]   // 一整个
```

## slice 的长度和容量

- 通过 `len(slice)` 获取其长度
- 通过 `cap(slice)` 获取其容量
- 通过再次切自己来移除前后的元素
- 通过再次切自己来延长（如果后面还有容量的话，超出容量会 panic）
```go
	s := []int{2, 3, 5, 7, 11, 13}
	printSlice(s)

	// 切成长度 0
	s = s[:0]

	// 延长长度为 4 (0, 1, 2, 3) 
	s = s[:4]

	// 移除开头的两个元素
	s = s[2:]
```

## 空 slice

未初始化的 slice 的值为 `nil`，长度和容量均为 0，且没有底层的数组

## 用 make 初始化指定长度和容量的 slice

```go
a := make([]int, 5)  // len(a)=5
b := make([]int, 0, 5) // len(b)=0, cap(b)=5
```

- `make([]int, 0, 5)` = "我要一个空的容器，但预留 5 个位置"
- `make([]int, 5, 5)` = "我要一个已有5个零值元素的切片"（append 时会从第 6 个开始）

## 嵌套 slice

```go
	board := [][]string{
		[]string{"_", "_", "_"},
		[]string{"_", "_", "_"},
		[]string{"_", "_", "_"},
	}
```

## append

go 有一个内置函数`append`，能向 slice 追加元素（不改变原 slice，返回一个新的 slice）

```go
func append(s []T, vs ...T) []T
```

如果添加元素后，底层数组不够大，那么会新分配一个数组，新 slice 会指向那个新分配的数组
