# font-
css中，字体可由以下几个属性控制：

## font-family

用于设置字体名称，可以指定多个字体作为备选，用逗号分隔。

```css
font-family: "Microsoft YaHei", "PingFang SC", sans-serif;
```

- 字体名称包含空格时需要用引号包裹
- 最后通常指定一个通用字体族（如 `serif`、`sans-serif`、`monospace`）作为兜底

## font-size

用于设置字体大小。

```css
font-size: 16px;
font-size: 1.2em;
font-size: 1.5rem;
font-size: 14pt;
```

常用单位：
- `px`：像素，绝对单位
- `em`：相对于父元素的字体大小
- `rem`：相对于根元素（html）的字体大小
- `%`：相对于父元素的百分比
- `pt`：磅，印刷单位

## font-weight

用于设置字体粗细。一般的字体只有normal和bold，少数专业设计字体有其他选项。

```css
font-weight: normal;    /* 400 */
font-weight: bold;      /* 700 */
font-weight: 400;
font-weight: 700;
```

常用值：
- `normal`（400）：正常粗细
- `bold`（700）：加粗
- `100` ~ `900`：数值表示，100 最细，900 最粗
- `lighter`：比父元素更细
- `bolder`：比父元素更粗

## font-style

用于设置字体样式。

```css
font-style: normal;     /* 正常 */
font-style: italic;     /* 斜体 */
font-style: oblique;    /* 倾斜 */
```

- `italic`：使用字体的斜体版本
- `oblique`：将正常字体倾斜显示

## font-variant

用于设置小型大写字母。

```css
font-variant: normal;
font-variant: small-caps;
```

## font-stretch

用于设置字体的拉伸/压缩程度（选择字体中的窄体或宽体版本）。

```css
font-stretch: normal;       /* 正常 */
font-stretch: condensed;    /* 紧缩 */
font-stretch: expanded;     /* 加宽 */
font-stretch: 75%;          /* 百分比值 (50% - 200%) */
```

常用关键字值：
- `ultra-condensed`：最紧缩
- `extra-condensed`：非常紧缩
- `condensed`：紧缩
- `semi-condensed`：略紧缩
- `normal`：正常（默认）
- `semi-expanded`：略加宽
- `expanded`：加宽
- `extra-expanded`：非常加宽
- `ultra-expanded`：最加宽

> **注意**：只有当字体本身提供了对应宽度的变体时，`font-stretch` 才会生效。大多数中文字体不支持此属性。

## line-height

用于设置行高。

```css
line-height: 1.5;
line-height: 24px;
line-height: 150%;
```

- 无单位数值：相对于字体大小的倍数（推荐）
- 带单位：绝对值或相对于字体大小的百分比

# font

上述的几个属性可以简写在单个 `font` 属性中，注意必须按照指定顺序：

```css
font: font-style font-variant font-weight font-stretch font-size/line-height font-family;
```

**必须包含 `font-size` 和 `font-family`**，其他属性可选。

```css
/* 完整写法 */
font: italic small-caps bold 16px/1.5 "Microsoft YaHei", sans-serif;

/* 简洁写法 */
font: 14px/1.6 "PingFang SC", sans-serif;

/* 使用系统默认字体 */
font: 16px -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif;
```

# @font-face

用于引入自定义字体文件。

```css
@font-face {
  font-family: "MyFont";
  src: url("myfont.woff2") format("woff2"),
       url("myfont.woff") format("woff");
  font-weight: normal;
  font-style: normal;
  font-display: swap;
}

body {
  font-family: "MyFont", sans-serif;
}
```

常用字体格式：
- `woff2`：压缩率最高，现代浏览器支持
- `woff`：广泛支持
- `ttf`：TrueType 字体
- `otf`：OpenType 字体
- `eot`：IE 专用格式

`font-display` 属性：
- `auto`：浏览器默认行为
- `swap`：先显示备用字体，加载完成后再切换（推荐）
- `block`：短暂隐藏文字，字体加载后显示
- `fallback`：短暂隐藏，如果加载慢则使用备用字体
- `optional`：与 fallback 类似，但可能完全不加载字体

# 常用系统字体栈

```css
/* 中文无衬线 */
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, "Noto Sans", "PingFang SC", "Microsoft YaHei", sans-serif;

/* 中文衬线 */
font-family: "Songti SC", "SimSun", "STSong", serif;

/* 等宽字体 */
font-family: "SF Mono", Monaco, "Cascadia Code", "Roboto Mono", Consolas, "Courier New", monospace;
```
