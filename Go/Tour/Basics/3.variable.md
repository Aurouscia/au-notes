# 变量

变量通过 `var` 关键字定义
- 可一次性定义多个（逗号分开）
- 类型标注跟在名称后面（空格分开）
- 多个类型一致的只需要写最后一个类型标注
- 变量可以是包级的或函数级的

```go
package main

import "fmt"

var c, python, java bool // 包级变量

func main() {
	var i int // 函数级变量
	fmt.Println(i, c, python, java)
}
```

# 初始化

使用 `=` 初始化变量值
- 有初始化值时，不需要类型标注（可直接使用值的类型）
- 多个变量可使用逗号分隔的值一起初始化，每个变量使用对应位置的值

```go
var i, j int = 1, 2
var c, python, java = true, false, "no!"
```

# 简短初始化

使用 `:=` 声明+初始化变量
- 仅在函数内可用，因为函数外的语句必须以关键字开头

```go
func main(){
    x := 1
}
```

# 使用括号的初始化

```go
package main

import "math/cmplx"

var (
	ToBe   bool       = false
	MaxInt uint64     = 1<<64 - 1
	z      complex128 = cmplx.Sqrt(-5 + 12i)
)
```

# 常量

不可修改版的变量，只能使用 `const` 关键字定义，不能用 `:=`

```go
const x = 1
```

## 数值常量

数值常量可以实现极大的范围

```go
const (
	Big = 1 << 100 // 可以创建 1 左移 100 位的巨大数字
	Small = Big >> 99
)
```
