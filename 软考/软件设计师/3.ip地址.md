# 子网掩码
子网掩码：用来标识一个IP地址的网络部分和主机部分。
子网掩码是一个32位的二进制数，其中网络部分为1，主机部分为0。

| IP地址类别 | 默认子网掩码 | CIDR表示法 |
| ------  | ------------- | ------- |
| A类     | 255.0.0.0     | /8      |
| B类     | 255.255.0.0   | /16     |
| C类     | 255.255.255.0 | /24     |

例：
- 地址块“/26”表示前26位为网络部分，后6位为主机部分，也就是有 2^6=64个 主机地址。
- 地址块“/19”表示前19位为网络部分，所以里面包含 2^(24-19)=32 个C类网络
