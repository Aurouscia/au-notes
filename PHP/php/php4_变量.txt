getType() 获取变量的类型名称，返回字符串
var_dump() 输出变量的完整信息
setType(变量名,目标类型)改变一个变量的类型
（区别于强制转换只转换里面的值，不改变本身）
isset() 判断一个变量是否存在（区别于常量的defined）（严格）
empty() 判断一个变量是否为空（不严格）
unset() 销毁一个变量
is_int() is_float() is_numeric()(包括纯数字字符串)
is_scalar()(是否为int,float,string,bool) 判断某数据是否是某类型

数据类型：
标量：int float string bool
复合：array object
特殊：null resource

整数类型
三种写法：
十进制写法：直接写
八进制写法：前面加个0
16进制写法：前面加个0x
$n3=0x123
decbin()十进制转二进制
decoct()十进制转八进制
dechex()十进制转16进制
输入十进制数字，输出是其他进制字符串
bindec("二进制字符串")，
输入是其他进制字符串，输出是是十进制数字

浮点数类型
float double real
$v1 = 1.234，带小数点，就是浮点数
$v2 = 1.234E3 即为1.234*10^3
$v3 = 1234E3 也是浮点数
可能比较接近的浮点数不应该进行大小比较，因为不精确
解决方法，*1000(要求三位)然后round()之再比较
整数超出范围后会自动转换为浮点数

字符串类型
js中虽然有两种表达形式，但是理解为同一种字符串
var str1='单引号字符串';
var str2="双引号字符串";
PHP中单引号和双引号字符串，是有不同细节含义的
有四种字符串形式
单引号字符串'xxxx'
可以识别的转义符有：\\   \'
（非转义字符写啥就是啥）
双引号字符串"xxxx"
可以识别的转义字符：\\   \"   \n   \r   \t   \$
因为【双引号字符串中会自动识别变量】所以$必须用转义字符
nowdoc(单引号)定界符
$str3=<<<'ABC'
xxxxxxxxx
(可以写任何东西都行，包括回车，$，没有可用的转义)
(通常用来写大段的html东西和js代码)
ABC;
说明：ABC是个自定义的标识符，放在单引号里
以同样的东西（结束标识符）结尾【这一行只能有结束标识符和分号，空格也不行】
heredoc(单引号)定界符
$str4=<<<"ABCD"
xxxxxxxxxx
这里需要用转义字符，才能打特殊符号(双引号不用)
可以识别变量
ABCD;

布尔
用于标识只有两种状态的数据
应用中会把数据直接当布尔值用
隐式转换会转换为false的有：
0 , 0.0 , null , "" , "0" , array(0) (不包括未定义，那是语义上的错误)
其他数据就当作true了

数组
标识一系列有序排列的数据的集合
数组的下标可以使用整数(索引号)或字符串(键名)，叫index或offset
数组里存储数组，叫多维数组
$arr1=array(5,22,12,'d1'=>'abc')
foreach(数组名 as 下标变量$key => 值变量$value){
     echo "<br/> 下标：$key ，值：$value";
}

对象类型 object
和js的区别比较大，js里有自定义和现成的window,location等
php的对象都是自己定义的
class Person{
    public $name="xxx";
    public $age=18;
    public function intro(){
        echo "hello!";
        echo "我叫"  . $this->name;
    }
}
$person1 = new Person();//创建了一个对象类型
$person1->name="小明";
$person1->age=22;
$person1->intro()

资源类型 resource
对外部数据的引用，不能通过php创建，是本来就有的

空类型 null
没有数据，没有有效的有意义的数据

自动转换：
if()：转为bool类型
算数运算符：转为数字类型
连接运算符：转为字符串类型
比较运算符：转为布尔或者数字类型
（两边有一边是布尔，另一边就会转为布尔，否则就执行数字比较）
字符串转为数字，是从左到右扫描，碰到第一个不是数字和小数点的字符就停止
强制转换
(int)7.8 将浮点数转为整数，截断

可变变量
一个变量的名本身也是一个变量
$v1="abc" //这是个字符串，内容是"abc"
$abc=10 //这是个普通变量，内容是数字10
echo $$v1
一个$符号出现，就视图解释为一个变量，变量名为$v1，
其为abc，所以整体为$abc，值为10
