# 考试 1：变量与常量概念题

## 一、填空题

1. Go 中声明变量的关键字是 ______，声明常量的关键字是 ______。
    - var
    - const

2. 短变量声明 `:=` 只能在 ______（函数内/函数外/任何地方）使用。
    - 函数内

3. 未初始化的 `int` 类型变量的默认值（零值）是 ______， `bool` 类型的零值是 ______， `string` 类型的零值是 ______。
    - 0
    - false
    - ""

4. `iota` 在 `const` 块中从 ______ 开始递增。
    - 0

5. 常量声明时 ______（必须/可以不必）初始化，变量声明时 ______（必须/可以不必）初始化。
    - 必须
    - 可以不必

## 二、判断题（正确填✓，错误填✗）

1. （ ）`var a int = 10` 和 `a := 10` 完全等价。
    - 对

2. （ ）全局变量可以使用 `:=` 声明。
    - 错，只能在函数内用

3. （ ）常量的值在运行时可以修改。
    - 错

4. （ ）`const Pi = 3.14` 中，Pi 的类型是 float64。
    - 对

5. （ ）`x, y := 10, 20` 声明了两个变量 x 和 y。
    - 对

6. （ ）以下代码是合法的：
   ```go
   x := 10
   x, y := 20, 30
   ```
    - 错，x 重复声明

7. （ ）常量可以使用函数的返回值进行初始化。
    - 错，必须编译期确定

## 三、代码分析题

### 第 1 题

```go
package main

import "fmt"

const (
    A = iota
    B
    C
)

func main() {
    fmt.Println(A, B, C)
}
```

输出结果是：
- 0，1，2

### 第 2 题

```go
package main

import "fmt"

func main() {
    var a int
    var b string
    var c bool
    fmt.Printf("%d|%s|%t", a, b, c)
}
```

输出结果是：
- 0|""|false

### 第 3 题

```go
package main

func main() {
    const (
        KB = 1 << (10 * iota)
        MB
        GB
    )
    println(KB, MB, GB)
}
```

输出结果是：
- 1，1024，1048576
