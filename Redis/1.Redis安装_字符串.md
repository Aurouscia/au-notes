# 基本操作
在ubuntu控制台中输入redis-cli，即可进入redis交互命令行

# 基本命令
## GET/SET
    set key value
    get key
### 覆盖？
    set key value nx 只在key不存在时成功
    set key value xx 只在key存在时成功
### GETSET
    设置一个新值并返回旧值
### 多个
    mset a 10 b 20 c 30
    mget a b c
### INCR
    让一个数字自增1
    如果是字符串表示的数字，会先parse之后自增1然后转回字符串
#### 类似命令
    INCRBY key increment
    DECR
    DECRBY


## Key
可以使用任何二进制序列作为key
### 正确：
    user:1000:followers
    comment:4321:reply-to
### 错误：
    u1000flw
    cmt4321r
### 大小限制
    key和value都限制512MB大小
### 看某key是否存在
    exists keyName
    返回1/0
### 删除键值对
    del keyName
    返回1/0（是否存在keyName）

## key过期
可以设置某key的数据一段时间后自动删除
Time To Live:TTL
设置key五秒后过期
### 方法1
    expire key 5
### 方法2
    set key 100 ex 5
### 查看多长时间后过期
    ttl key
### 相关命令
    PEXPIRE
    PTTL
