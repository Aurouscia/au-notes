# 考试 2：流程控制实践题

## 需求

创建一个 Go 程序，实现一个数字猜谜游戏。

### 项目结构

```
exam2-practice/
├── go.mod
└── main.go
```

### 具体要求

#### 1. 初始化模块

```bash
go mod init exam2-practice
```

#### 2. 实现猜谜游戏（main.go）

程序需要实现以下功能：

1. **生成随机数**
   - 使用 `rand.Intn(100)` 生成 0-99 之间的随机数作为目标数字
   - 使用 `rand.Seed(time.Now().Unix())` 设置随机种子（需要导入 `math/rand` 和 `time` 包）

2. **游戏循环**
   - 使用无限循环 `for { ... }` 让玩家可以多次猜测
   - 每次循环中：
     - 提示玩家输入一个数字
     - 使用 `fmt.Scan(&guess)` 读取玩家输入
     - 判断猜测结果：
       - 猜对了：输出"恭喜你猜对了！"，并结束游戏
       - 猜大了：输出"太大了，再试一次"
       - 猜小了：输出"太小了，再试一次"

3. **限制猜测次数**
   - 最多允许猜测 7 次
   - 使用 `for i := 0; i < 7; i++` 替代无限循环
   - 如果 7 次都没猜对，输出"游戏结束，正确答案是：X"

4. **统计功能**（选做）
   - 记录玩家猜测的历史数字
   - 游戏结束后，使用 `for-range` 遍历并打印所有猜测过的数字

### 预期输出示例

```
欢迎来到猜数字游戏！数字范围是 0-99，你有 7 次机会。
请输入你的猜测：50
太大了，再试一次
请输入你的猜测：25
太小了，再试一次
请输入你的猜测：37
恭喜你猜对了！
```

或（7次未猜中）：

```
欢迎来到猜数字游戏！数字范围是 0-99，你有 7 次机会。
请输入你的猜测：10
太小了，再试一次
...
游戏结束，正确答案是：73
你猜测过的数字：[10 20 30 40 50 60 70]
```

### 提示

1. 导入需要的包：
   ```go
   import (
       "fmt"
       "math/rand"
       "time"
   )
   ```

2. 设置随机种子：
   ```go
   rand.Seed(time.Now().Unix())
   ```

3. 生成随机数：
   ```go
   target := rand.Intn(100)  // 0-99
   ```

4. 读取用户输入：
   ```go
   var guess int
   fmt.Scan(&guess)
   ```
