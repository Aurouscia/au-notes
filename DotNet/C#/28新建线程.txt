【Thread类】
Thread.Sleep(毫秒数)  等待一段时间
Thread.CurrentThread.ManagedThreadId  取得当前线程id

构造时为线程指定一个函数
using System.Threading
Thread t=new Thread(函数名);
t.Start();

或者用lambda表达式写个匿名函数：
Thread t=new Thread(()=>{函数体});  （空括号，因为没有参数）
t.Start

【可以有一个参数，可以没有】参数必须是object类型
static void DownloadFile(object fileName){xxx;}

【任务】
Task t=new Task(Action或者方法隐式转换为Action)
t.Start()
貌似不能有参数

TaskFactory tf=new TaskFactory();

连续任务：可以控制先后关系
Task t1=new Task(Test1);
Task t2=t1.ContinueWith(Test2);
t1做完之后才去做t2
Task t3=t1.ContinueWith(Test2);

任务层次结构：
在任务里面新开一个任务，就是子任务，异步执行
（不同于线程，线程没有父子关系）
父任务有个WaitingForChildrenToComplete状态
子任务完事后会变成RunToCompletion
