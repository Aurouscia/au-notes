定义组件的函数，可以有唯一一个参数props，props是一个对象，包含了组件的属性
```tsx
export function TodoItem1(props) {
    // props中的属性可以通过props.属性名来访问
    return ();
}
```

也可以直接用解构赋值的方式来访问属性，可以写默认值，例如 = false
```tsx
export function TodoItem2({ title, completed = false }) {
    return ();
}
```

如果组件本身不使用属性，而是直接传给子组件，那么可以使用展开运算符来传递属性，以简化代码
```tsx
export function Profile(props) {
  return (
    <div className="card">
      <Avatar {...props} />
    </div>
  );
}
```

组件属性可以是一个子组件，这时props中会有一个名为children的属性，它包含了组件的子组件
//相当于vue里的slot
```tsx
function Card({ children }: { children: React.ReactNode, otherProp?: string } ) {
  return (
    <div className="card">
      {children}
    </div>
  );
}
export default function Profile() {
  return (
    <Card>
      <Avatar size={100}/>
    </Card>
  );
}
```
也可以用以下方法定义类型：
```tsx
interface Card2Props{
    children:React.ReactNode,
    otherProp?:string,
}
//或：
type Card2Props = React.PropsWithChildren<{
otherProp?:string,
}>
```
注意：
props是不可变的（除非使用state来存储props）
