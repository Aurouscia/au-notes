# 函数

## 定义和调用
```bash
# 基本函数语法
chpwd() {
    # 函数体
    local need16_prefix='/Users/au/yh-projects'
    # ...更多代码
}

# 调用函数
chpwd
```

## 函数调用和参数
- 函数调用不要括号，括号会被当作命令执行
- 函数的参数直接写在函数名后面，用空格分隔，例如`greet $NAME1 $NAME2`
- 函数内通过`$1`、`$2`等变量获取参数
    - `$#`：获取参数个数
    - `$*`：获取所有参数
    - `$0`：获取当前脚本名（不是函数名）
    - `$@`：获取所有参数（与`$*`类似，但会将每个参数作为独立的字符串）

## 特殊函数
### chpwd函数
- zsh中的特殊钩子函数，当改变当前目录时自动执行
- 无需手动设置，定义即生效
