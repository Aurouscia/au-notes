# 指针

go 中有指针类型，`var p *int`代表一个指向整数的指针

- *p 间接访问符：通过一个指针获取其底层的值 dereferencing
- &x 取地址符：获取一个指向某值的指针 indirecting

```go
func main() {
	i, j := 42, 2701

	p := &i         // 创建指向 i 的指针 p
	fmt.Println(*p) // 通过指针 p 来读取 i
	*p = 21         // 通过指针 p 来设置 i
	fmt.Println(i)  // 看看 i 的新值

	p = &j         // 改为指向 j
	*p = *p / 37   // 进行算数操作，读取和赋值
	fmt.Println(j) // 看看 j 的新值
}
```
