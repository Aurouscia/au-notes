# css选择器

## 子选择器
`.box>.classname`表示.box下类名为classname的元素(仅一代，孙子辈不行)

## 包含选择器
`#header p`表示header下的p元素(所有内部的都包括，不管几代)

## 相邻选择器
表示选择某个元素后面的第一个元素，`div+p`表示div后面紧跟的p

## 选择器的组合 
`p.classname{xx}`可以更准确
`div#box`会产生比`#box`更高的优先级

## 选择器可以连缀
例如`#wrap #header h2 span`连着写四个，中间用空格隔开，表示三重包含关系

## 属性选择器
`img[alt]`表示含有alt属性值的img元素
`img[alt="114514"]`表示含有alt而且其值为114514的元素
可以连着写几个`img[title="xx"][alt="xx"]`

## 模糊式属性选择器
把属性选择器里的=换成|=,~=等等
例如img[class|="blue"]
- |=以连字符分隔，匹配分隔后的小字符串
- ~=以空格分隔，匹配分隔后的小字符串
- ^=前缀匹配，从开头匹配
- $=后缀匹配，从结尾匹配
- *=子字符串匹配，在里面任意地方都行

## 样式群
选择器之间用逗号隔开，可以对数类不同的元素定义一样的样式
`h1, h2, h3, h4{xxx}`
