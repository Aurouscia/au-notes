## 二、判断题（请在每题后写出判断结果和理由）

1. Go 语言函数默认使用引用传递，函数内对参数的修改会影响原变量。

2. 以下函数定义是合法的：
   ```go
   func calculate(x int, y int) (sum int, diff int) {
       sum = x + y
       diff = x - y
       return
   }
   ```

3. 命名返回值在函数开始执行时会被初始化为对应类型的零值。

4. 函数必须至少有一个返回值。

5. 以下代码中，`main` 函数打印的 `x` 和 `y` 值会被交换：
   ```go
   func swap(a, b int) {
       a, b = b, a
   }
   
   func main() {
       x, y := 1, 2
       swap(x, y)
       fmt.Println(x, y)
   }
   ```
