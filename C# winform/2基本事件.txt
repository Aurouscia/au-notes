双击编辑器的窗体
就会打开该控件的代码文件
并自动添加一个load函数，注册到Load事件里面
（已有也会重复添加）

public Form1()  //该窗体对象的构造函数
{
   InitializeComponent();//自动代码，不要动这里
   //这里面包括了窗体绑定事件

   //写声明在这里可能会引起顺序错乱，不要写

   //可以为load事件加入函数，顺序排在窗体绑定之后
   this.Load+=Form1_Load
}

“属性”里的闪电图标处可以注册事件
或者双击创建一个相应事件的函数
其中Load事件会在第一次显示窗体前，Init后发生

(小字)1个引用
private void Form1_Load(object sender,EventArgs e)
{
   //走完了上面的Form1()之后会执行load事件，
   //load事件一般绑定了此方法
   //也可以手动绑定（见上面）
   this.Text="这是由窗体Load事件修改的标题 "
}




点击关闭窗口键后
Formclosing立即执行
然后是Formclosed
如果用进程管理器之类的东西关掉窗口，是不会调用的
这里只能处理内部（叉叉键）消息

如果要清理，销毁对象，全放在closing里面更合适
