【创建object实例】
【自定义的对象，可以加任意属性和方法】
1.使用new和Object构造函数
var obj=new Object();
obj.name="xxx";会自动增加属性
obj.school="xxx";
obj.showInfo=function(xx){xxx;};
增加一个方法，左边不需要括号（表示函数本身而不是返回值）右边是函数定义
使用时：
document.write(obj.name);
obj.showInfo(xx);
也可以写：obj['name']访问
2.使用大括号
var obj={name:"xx",school:"xx",showInfo:getInfo（不要括号，表示函数本身而非其返回值）}

【类】
1.用function写个构造函数
function myclass(){
    this.id=5;
    this.name='myclass';
    this.getName=function(){return this.name};
}
var my=new myClass();
2.在函数中声明对象并添加属性，然后返回
function myClass(){
    var obj={'id':2,'name':'myclass'}
    return obj;}
var my=new myClass();
【创建多个类似的自定义对象】
function createCar(color)
    var tempCar=new Object;
    tempCar.color=color;
    tempCar.showColor=function(){alert(this.color)}（也可以把同样的函数提到外面避免重复）
    return tempCar;}

1.使用new和String构造函数

string对象的方法
1.charAt(index)方法，查找特定索引号的字符
	var char=xxstr.charAt(0);
2.indexOf(char)方法，从左到右查找子字符串位置
3.lastIndexOf(char)方法，从右到左
4.substr(startIndex,length（可选）)提取子字符串
5.substring(startIndex,endIndex)
6.split(分割符)返回一个array
7.toLower/UpperCase(不填)
8.localeCompare()，用当地语言的排序方式比较大小
	var arr=["xxx","xxx","xxx","xxx"];
	arr.sort(orderby2);（提供应该比较方法）
	function orderby2(str1,str2)
		{return str1.localeCompare(str2);}

回忆：meta标签里的refresh属性
不适合做时钟，因为刷新整个网页十分耗费资源
应该用：window.setInterval(JS代码，毫秒数)
【动画就是用这个做的】
window.setTimeout(JS代码(一般是个函数)，毫秒数)
window.setInterval(JS代码，毫秒数)
前者：延迟一段时间后执行一次
后者：隔一段时间执行一次

function editBg(){（返回特定最大值和最小值之间的随机数）
    var min=100000;
    var max=999999
    var color=Math.floor(Math.random()*(max-min)+min);
    document.bgColor="#"+color;
}
