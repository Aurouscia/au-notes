# Linux 用户系统

## useradd - 添加用户

```bash
useradd 用户名
```

### 常用选项

| 选项 | 全称 | 说明 |
|------|------|------|
| `-g` | `--gid` | 指定主用户组 |
| `-G` | `--groups` | 指定附加组 |
| `-d` | `--home` | 指定家目录 |
| `-M` | `--no-create-home` | 不创建家目录 |
| `-c` | `--comment` | 添加注释/描述 |
| `-u` | `--uid` | 指定用户 ID |
| `-s` | `--shell` | 指定登录 shell |
| `-e` | `--expiredate` | 设置账户过期日期 |

### 示例

```bash
# 指定主用户组
useradd -g group1 用户名

# 指定家目录（默认是 /home/用户名）
useradd -d /home/user1 用户名

# 不创建家目录
useradd -M 用户名

# 添加注释
useradd -c "Test User" 用户名

# 指定用户 ID
useradd -u 1001 用户名

# 指定登录 shell
useradd -s /bin/bash 用户名

# 组合使用
useradd -u 1001 -g developers -G sudo,docker -d /home/devuser -s /bin/bash -c "Developer" devuser
```

## passwd - 设置密码

```bash
# 给指定用户设置密码（需要 root 权限）
passwd 用户名

# 修改自己的密码
passwd

# 锁定用户密码
passwd -l 用户名

# 解锁用户密码
passwd -u 用户名

# 删除用户密码
passwd -d 用户名

# 显示密码状态
passwd -S 用户名
```

## usermod - 修改用户

```bash
usermod [选项] 用户名
```

> 选项与 `useradd` 基本一致

### 常用操作

```bash
# 将用户添加到附加组（保留原有组）
usermod -aG 组名 用户名

# 修改主组
usermod -g 新主组 用户名

# 修改家目录
usermod -d /new/home/dir -m 用户名

# 修改用户名
usermod -l 新用户名 旧用户名

# 锁定账户
usermod -L 用户名

# 解锁账户
usermod -U 用户名
```

> **注意**：每个用户有一个**主组（GID）**和多个**附加组**，一个用户可以存在于多个组里

## userdel - 删除用户

```bash
# 仅删除用户
userdel 用户名

# 删除用户及其家目录和邮件池
userdel -r 用户名

# 强制删除（即使用户已登录）
userdel -f 用户名
```

## id - 查看用户信息

```bash
# 查看指定用户的 UID、GID 和所属组
id 用户名

# 仅查看 UID
id -u 用户名

# 仅查看 GID
id -g 用户名

# 查看所有组 ID
id -G 用户名
```

输出示例：
```
uid=1001(devuser) gid=1001(developers) groups=1001(developers),27(sudo),999(docker)
```

## 用户组管理

### groupadd - 添加组

```bash
# 创建新组
groupadd 组名

# 指定 GID
groupadd -g 1001 组名
```

### groupmod - 修改组

```bash
# 修改组名
groupmod -n 新组名 旧组名

# 修改 GID
groupmod -g 新GID 组名
```

### groupdel - 删除组

```bash
groupdel 组名
```

> 不能删除用户的主组

### gpasswd - 管理组成员

```bash
# 将用户添加到组
gpasswd -a 用户名 组名

# 从组中移除用户
gpasswd -d 用户名 组名

# 设置组管理员
gpasswd -A 用户名 组名
```

## 切换用户

```bash
# 切换到 root 用户
su -

# 切换到指定用户（加载其环境变量）
su - 用户名

# 切换到指定用户（保留当前环境变量）
su 用户名

# 执行命令后返回
su - 用户名 -c "命令"
```

## sudo 权限配置

```bash
# 编辑 sudoers 文件
visudo
```

### 常用配置

```bash
# 给用户添加 sudo 权限
用户名 ALL=(ALL:ALL) ALL

# 给组添加 sudo 权限
%组名 ALL=(ALL) ALL

# 免密码 sudo
用户名 ALL=(ALL) NOPASSWD: ALL

# 限制特定命令
用户名 ALL=(root) NOPASSWD: /usr/bin/systemctl restart nginx
```

## 相关文件

| 文件 | 说明 |
|------|------|
| `/etc/passwd` | 用户账户信息 |
| `/etc/shadow` | 用户密码（加密存储）|
| `/etc/group` | 组信息 |
| `/etc/gshadow` | 组密码 |
| `/etc/sudoers` | sudo 权限配置 |
| `/etc/login.defs` | 用户账户创建默认值 |
| `/etc/default/useradd` | useradd 默认值 |
