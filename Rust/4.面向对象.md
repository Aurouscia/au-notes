# 面向对象

## struct 定义数据
```rs
struct User {
    name: String,
    age: u32,
}
```

## impl 定义方法
- impl 块能把函数“挂”到某个类型上：
- 方法（method）：第一个参数是 `self`、`&self` 或 `&mut self`，通过实例调用。
- 关联函数（associated function）：没有 `self`，通过 `Type::func()` 调用，常用作构造函数。
```rs
impl User {
    // 构造函数（约定叫 new）
    fn new(name: String, age: u32) -> Self {
        User { name, age }
    }
    
    // 实例方法（&self 类似 this，&表示只读借用，不发生所有权转移）
    fn greet(&self) {
        println!("Hello, I'm {}", self.name);
    }
    
    // &mut表示可写借用，也不发生所有权转移，但能修改
    fn birthday(&mut self) {
        self.age += 1;
    }
}
```

## trait 定义特征
```rs
trait Drawable {
    fn draw(&self);
    fn description(&self) -> String {
        "a drawable object".to_string()  // 默认实现
    }
}

// 为 struct 实现 trait（可覆盖trait中默认实现）
impl Drawable for User {
    fn draw(&self) {
        println!("Drawing user: {}", self.name);
    }
}
```

## 函数实现和调用
UCS (universal call syntax)
`<trait名>::<方法名>()`

### impl实现（覆盖）trait的方法
- 方法：有第一个参数为self的函数
- 固有impl：名称与struct一样的impl（调用方法时优先级最高）
```rs
struct S;
impl S {
    fn foo(&self) { println!("inherent") }
}
trait Tr {
    fn foo(&self) { println!("trait") }
}
impl Tr for S {}

let s = S;
s.foo();        // 默认：调用impl中实现的方法
Tr::foo(&s);    // 也可手动指定trait中原本的方法
```

### 调用不同trait的同名方法
```rs
trait Bark {
    fn noise(&self) -> &'static str { "woof" }
}
trait Meow {
    fn noise(&self) -> &'static str { "meow" }
}

struct Animal;
impl Bark for Animal {}
impl Meow for Animal {}

let a = Animal;
// 默认无法直接 a.noise()，因为编译器不知道要哪个 trait
// 用 UCS 显式指定
println!("{}", Bark::noise(&a)); // woof
println!("{}", Meow::noise(&a)); // meow
```

## 构造函数
- `Self`（大写）是类型别名，指代“当前这个 impl 块所实现的类型本身”
- `self`（小写）是变量名，指代“当前这个方法收到的实例（或引用）”
```rs
struct Rectangle { w: u32, h: u32 }
impl Rectangle {
    // 返回值类型写成 Self，等价于 Rectangle
    fn new(w: u32, h: u32) -> Self {
        Self { w, h }   // 这里 Self 也是类型，省去重复写 Rectangle
    }
    // 关联函数也能拿 Self 做参数
    fn from_square(size: u32) -> Self {
        Self { w: size, h: size }
    }
}
```
