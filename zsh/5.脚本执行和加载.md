# 脚本执行和加载

## source命令
```bash
# 重新加载配置文件
alias rc="source ~/.zshrc"

# 按条件（是否存在）加载外部脚本
[ -s "$nvm_sh" ] && source "$nvm_sh"
[ -s "$nvm_completion" ] && source "$nvm_completion"
```

## 脚本结构示例
```bash
# NVM配置示例
export NVM_DIR="$HOME/.nvm"                    # 设置环境变量
nvm_sh="/opt/homebrew/opt/nvm/nvm.sh"         # 定义脚本路径
[ -s "$nvm_sh" ] && source "$nvm_sh"         # 条件加载主脚本
nvm_completion="/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm"  # 补全脚本路径
[ -s "$nvm_completion" ] && source "$nvm_completion"              # 条件加载补全脚本
```
