# Go 语言数组（Array）

## 1. 什么是数组

数组是**固定长度**的、**同类型**元素的有序集合。数组的长度是类型的一部分，因此 `[3]int` 和 `[5]int` 是不同的类型。

## 2. 数组的声明与创建

### 2.1 基本声明

```go
// 声明一个长度为5的int数组
var arr1 [5]int
// 默认值：[0, 0, 0, 0, 0]

// 声明并初始化
var arr2 = [3]int{1, 2, 3}

// 短变量声明
arr3 := [4]string{"Go", "Python", "Java", "Rust"}
```

### 2.2 自动推断长度

使用 `...` 让编译器自动计算数组长度：

```go
arr := [...]int{10, 20, 30, 40}  // 编译器推断为 [4]int
```

### 2.3 指定索引初始化

```go
arr := [5]int{0: 100, 2: 200, 4: 400}
// 结果：[100, 0, 200, 0, 400]
```

## 3. 数组的基本操作

### 3.1 访问元素

```go
arr := [3]int{10, 20, 30}
fmt.Println(arr[0])  // 10
fmt.Println(arr[1])  // 20
```

### 3.2 修改元素

```go
arr := [3]int{10, 20, 30}
arr[1] = 99
// 结果：[10, 99, 30]
```

### 3.3 获取数组长度

```go
arr := [5]int{1, 2, 3, 4, 5}
fmt.Println(len(arr))  // 5
```

### 3.4 遍历数组

```go
arr := [3]string{"a", "b", "c"}

// 方式一：使用索引
for i := 0; i < len(arr); i++ {
    fmt.Println(arr[i])
}

// 方式二：使用 range（推荐）
for index, value := range arr {
    fmt.Printf("索引：%d，值：%s\n", index, value)
}
```

## 4. 数组的特性

### 4.1 值类型

**数组是值类型**，赋值或传递时会复制整个数组：

```go
arr1 := [3]int{1, 2, 3}
arr2 := arr1      // 复制整个数组
arr2[0] = 100

fmt.Println(arr1) // [1 2 3]（原数组不变）
fmt.Println(arr2) // [100 2 3]
```

### 4.2 数组比较

如果数组元素类型是可比较的，则数组也可以比较：

```go
a := [3]int{1, 2, 3}
b := [3]int{1, 2, 3}
c := [3]int{3, 2, 1}

fmt.Println(a == b) // true
fmt.Println(a == c) // false
```

## 5. 多维数组

```go
// 二维数组
matrix := [2][3]int{
    {1, 2, 3},
    {4, 5, 6},
}

// 访问元素
fmt.Println(matrix[0][1]) // 2
fmt.Println(matrix[1][2]) // 6

// 遍历二维数组
for i := 0; i < len(matrix); i++ {
    for j := 0; j < len(matrix[i]); j++ {
        fmt.Print(matrix[i][j], " ")
    }
    fmt.Println()
}
```

## 6. 数组的局限性

由于数组长度固定，在实际开发中更常用的是**切片（Slice）**。切片是基于数组的动态长度数据结构，将在后续章节介绍。

## 7. 小结

| 特性 | 说明 |
|------|------|
| 长度固定 | 声明后不能改变 |
| 类型的一部分 | `[3]int` 和 `[5]int` 是不同类型 |
| 值类型 | 赋值会复制整个数组 |
| 可比较 | 相同长度和元素类型的数组可以比较 |
| 索引访问 | 通过 `arr[index]` 访问元素 |
