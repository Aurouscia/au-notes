在jsx/tsx文件中，一个export JSX.Element的函数，可以作为一个组件使用
```tsx
export default function MyComp1() {
    return <div><h1>Hello World 1</h1></div>
}
```

定义组件的函数，必须以大写字母开头，例如MyComp1，MyComp2等  
这样使用的时候才能与html内置的标签区分开，例如<MyComp1/>，<MyComp2/>等

```tsx
export function MyComp2() {
    // react组件中，return后面可以是单行或多行jsx表达式，但多行的情况下，必须用()包裹起来
    return (
        <div>
            <h1>Hello World 2</h1>
        </div>
    )
}

export function TodoList() {
    // jsx中，单边标记必须闭合，例如“<img/>”，而不能写成“<img>”
    // return () 中的标签必须只有一个根标签，如果需要多个标签，可以使用<> </>包裹起来（相当于vue里的template）
    // 由于class是js中的关键字，所以在jsx中使用class时，需要使用className来代替
    return (
        <>
            <h1>海蒂·拉玛的待办事项</h1>
            <img
                src="https://i.imgur.com/yXOvdOSs.jpg"
                alt="Hedy Lamarr"
                className="photo"
            />
            <ul>
                <li>发明一种新式交通信号灯</li>
                <li>排练一个电影场景</li>
                <li>改进频谱技术</li>
            </ul>
        </>
    );
}
```
