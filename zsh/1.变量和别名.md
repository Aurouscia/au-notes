# 变量
- 本节内容`zsh`和`bash`等posix兼容的shell通用
- 变量名规则：
    - 不能以数字开头
    - 不能包含特殊字符，如空格、标点符号等
    - 区分大小写
- 赋值的等号`=`两侧不能有空格，如`my_name = Au`是错误的，应该是`my_name=Au`
- 字符串字面量
    - 一般情况下可省略引号
    - 当有空格等特殊字符时，必须使用引号括起来
    - **单引号**内的内容会**原样输出**，不进行变量替换
    - **双引号**内的内容会**进行变量替换**：例如`echo "Hello, $my_name"`会输出`Hello, Au`

## 变量定义
- 普通变量即为全局变量
- 引用时使用`$`符号，如`echo $my_name`
```bash
my_name=Au
my_email=18971501210@189.cn

echo $my_name
echo $my_email
```

## 局部变量定义
- 使用`local`关键字声明局部变量
- 局部变量只在当前函数或脚本中有效（离开后会被销毁）
```bash
local in_now=$PWD
local in_old=$OLDPWD
```

## 环境变量导出
- 使用`export`关键字导出环境变量
- 导出后，环境变量在子进程中也可用
```bash
export NVM_DIR="$HOME/.nvm"
```

## 特殊变量
- `$PWD` ：当前工作目录
- `$OLDPWD` ：上一个工作目录
- `$HOME` ：用户主目录

## 用命令替换赋值给变量
- 使用`$()`进行命令替换
- 小括号内的命令会被执行并将结果赋值给变量
```bash
# 使用$()进行命令替换，即执行小括号内命令并将结果赋值给变量
local name=$(git config --get user.name 2>/dev/null)
local email=$(git config --get user.email 2>/dev/null)
```

# 别名
- 使用`alias`关键字定义别名
- 别名可以简化命令，提高效率，例如zsh中可以定义其缺少的`ll`命令
- 别名定义后，在当前终端会话中有效
- 可以使用`alias`命令查看所有别名
- 可以使用`unalias`命令删除别名
```bash
alias ll="ls -l -a"
alias rc="source ~/.zshrc"
```
