# js中var和let的区别
## 变量提升机制
使用var声明的对象，都会被提升到该作用域的最顶部，例如：
```js
    console.log(a); //undefind
    var a = 1；
    console.log(a); //1
```
运行的时候就相当于
```
    var a;
    console.log(a); //undefined
    a = 1; 
    console.log(a); //1
```

let和const并没有这种机制（其前面有暂时性死区TDZ）

## 全局window对象
当var在顶级作用域使用时，变量会被直接加到window对象上，有可能会覆盖window本身的属性。

而let和const并不会

## 重复
let不允许再次声明一个已有的东西
