# 基本语法
`SELECT ..., <窗口函数调用> OVER (PARTITION BY <列名>)`  
在不改变原行数的情况下进行分组后有关计算，每行都得到一个结果（性能高于子查询）

# 例子
```sql
-- 按部门给员工按薪水降序排号
SELECT empno, deptno, sal,
       ROW_NUMBER() OVER (PARTITION BY deptno ORDER BY sal DESC) AS rn
FROM emp;
```

# 常用的窗口函数
- 排序类
    - ROW_NUMBER()  连续递增序号，不会重复
    - RANK()  并列名次会“跳号”（例如同一组的最高值rank都为1）
    - DENSE_RANK()  并列名次不跳号
- 错位类
    - LAG(col, n)  取当前行往上第 n 行的值（例如用户的上一次下单）
    - LEAD(col, n)  取当前行往下第 n 行的值
- 聚合类  
    SUM() / AVG() / COUNT() / MAX() / MIN()

- 取名次类
    - FIRST_VALUE(col) 取分区内第一行的值
    - LAST_VALUE(col) 取分区内最后一行的值
    - NTH_VALUE(col, n) 取分区内排序第 n 行的值
