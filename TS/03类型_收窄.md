# 收窄

## 类型收窄

ts分析器会自动根据代码的if-else结构判断变量在每一处的类型

<pre>
function padLeft(padding: number | string, input: string) {
  if (typeof padding === "number") {
    return " ".repeat(padding) + input;
  }
  return padding + input;
}
</pre>

获取变量的类型：使用js自带的typeof关键字

<pre>typeof padding</pre>

判断上述typeof的返回值将构成一个类型守卫（type guard）

<pre>typeof padding === "number"</pre>

## 布尔收窄
- 0
- NaN
- "" (the empty string)
- 0n (the bigint version of zero)
- null
- undefined

以上这些值在填到if里时都会被js运行时判断为false

**一种特殊情况：typeof null 会返回 "object"**

<pre>
function printAll(strs: string | string[] | null) {
  if (strs && typeof strs === "object") { //此处可以判断strs是不是null
    for (const s of strs) {
      console.log(s);
    }
  } else if (typeof strs === "string") {
    console.log(strs);
  }
}
</pre>

## in运算符收窄
<pre>
type Fish = { swim: () => void };
type Bird = { fly: () => void };
 
function move(animal: Fish | Bird) {
  if ("swim" in animal) {
    return animal.swim();
  }
 
  return animal.fly();
}
</pre>

## instanceof运算符收窄
<pre>
function logValue(x: Date | string) {
  if (x instanceof Date) {
    console.log(x.toUTCString());
               
(parameter) x: Date
  } else {
    console.log(x.toUpperCase());
               
(parameter) x: string
  }

## 自定义type guard
首先定义一个用来判断某对象是不是某个type的函数

注意冒号后面的写法
<pre>
function isFish(pet: Fish | Bird): pet is Fish {
  return (pet as Fish).swim !== undefined;
}
</pre>
<pre>
// Both calls to 'swim' and 'fly' are now okay.
let pet = getSmallPet();
 
if (isFish(pet)) {
  pet.swim();
} else {
  pet.fly();
}
</pre>

## 区分的联合类型

<pre>
interface Circle {
  kind: "circle";
  radius: number;
}
 
interface Square {
  kind: "square";
  sideLength: number;
}
 
type Shape = Circle | Square;
</pre>

当多个interface里面都有一个同名的字面量类型，
那么用它们组成的联合类型就可以通过if来收窄(通过判断那个字面量类型的值)

<pre>
function getArea(shape: Shape) {
  switch (shape.kind) {
    case "circle":
      return Math.PI * shape.radius ** 2;
    case "square":
      return shape.sideLength ** 2;
  }
}
</pre>

## never类型
当类型收窄到什么都不剩下的时候，这个变量就会被识别为never类型。

### 穷尽类型检查

<pre>
type Shape = Circle | Square;
 
function getArea(shape: Shape) {
  switch (shape.kind) {
    case "circle":
      return Math.PI * shape.radius ** 2;
    case "square":
      return shape.sideLength ** 2;
    default:
      const _exhaustiveCheck: never = shape;
      return _exhaustiveCheck;
  }
}
</pre>
如果写成这样，那么可以确保每一种情况都被处理了，如果Shape里添加了新的成员，那么这段代码就会报错。
