# case语句
```bash
case "$old_in_tree,$now_in_tree" in
    false,true)  nvm use 16      ;;   # 进入树
    true,false)  nvm use default ;;   # 离开树
esac
```

- 模式与命令之间用右括号`)`分隔
    - 例子中：如果第一个参数为false，第二个为true，则会匹配到第一个模式，执行其后面的命令
- 结尾的分号
    - `;;`表示“这个分支执行完就跳出 case，不再往下试别的模式”
    - `;&` 就会继续执行下一个分支的命令
    - `;;&` 会继续测试下一个模式。
