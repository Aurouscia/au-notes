# 组件的纯粹性
react组件必须是纯函数（即不依赖外部变量，不改变外部变量）  
否则会导致组件的状态无法正确更新，从而导致页面渲染错误。  
错误例子（不是纯函数）：
```jsx
function Cup() {
    // Bad：正在更改预先存在的变量！
    guest = guest + 1;
    return <h2>Tea cup for guest #{guest}</h2>;
}
```
这是由于：组件不一定出现几次调用几次，可能出现多次调用的情况。

没问题的例子（局部改变）：
```jsx
function Cup({ guest }) {
  return <h2>Tea cup for guest #{guest}</h2>;
}

export default function TeaGathering() {
  const cups = [];
  for (let i = 1; i <= 12; i++) {
    cups.push(<Cup key={i} guest={i} />);
  }
  return cups;
}
```

## 例外：事件处理
事件处理程序是 React 在你执行某些操作（如单击按钮）时运行的函数。即使事件处理程序是在你的组件**内部**定义的，它们也不会在渲染期间运行！ 因此事件处理程序无需是纯函数。

## 例外：useEffect
如果一定需要副作用，可以使用useEffect来实现。  
useEffect可以在组件渲染后执行副作用。
