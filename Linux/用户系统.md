# useradd
`useradd 用户名`

## 指定一个用户组
`useradd -g group1 用户名`

## 指定起始目录
`useradd -d /home/user1 用户名 `
（不设置的话默认是/home/用户名）

## 注释
`useradd -c "Test User" 用户名`

# passwd
`passwd 用户名`
给某用户设置密码，如果不填用户名就是设置自己的

# usermod
`usermod \[选项\] 用户名`
选项和useradd的一样

# userdel
删除用户

# uid
`id <username>`，查看某用户的 UID 和 GID（用户ID和组ID）
