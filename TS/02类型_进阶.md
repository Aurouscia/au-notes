# 联合类型(Union Type)

表示同一个参数可以传入多种类型（算是重载罢）

使用竖线（|）将类型隔开
```ts
function printId(id: number | string) {
  console.log("Your ID is: " + id);
}
// OK
printId(101);
// OK
printId("202");
// Error
printId({ myID: 22342 });
```

此时，id为string和number的量子叠加态

只接受string的函数不接受这种叠加态的联合类型参数

这种时候可以使用if else结构和typeof关键字进行"narrowing"，ts会自动判断每个分支里的类型

```ts
function printId(id: number | string) {
  if (typeof id === "string") {
    // In this branch, id is of type 'string'
    console.log(id.toUpperCase());
  } else {
    // Here, id is of type 'number'
    console.log(id);
  }
}
```

# 有名字的类型(type alias)

使用type关键字定义一个类型，并起名为Point

```ts
type Point = {
  x: number;
  y: number;
};
 
// Exactly the same as the earlier example
function printCoord(pt: Point) {
  console.log("The coordinate's x value is " + pt.x);
  console.log("The coordinate's y value is " + pt.y);
}
 
printCoord({ x: 100, y: 100 });
```

除了像上述那样给一个对象类型起名字之外，这种type也可以被赋值任何类型，比如说union

type ID = number | string;

## 不要以为是一个新版本
这种alias仅仅只是一个引用，它的对象和它表示的类型的对象完全可以互相赋值。

```ts
type UserInputSanitizedString = string;
 
function sanitizeInput(str: string): UserInputSanitizedString {
  return sanitize(str);
}
 
// Create a sanitized input
let userInput = sanitizeInput(getInput());
 
// Can still be re-assigned with a string though
userInput = "new input";
```

## typeAlias 和 interface的区别
interface可以很方便地继承和添加新字段，alias很难

# 类型断言(Assertion)
有些时候我很确定某个东西的类型，但ts推断不出来，这时候就可以使用断言

```ts
//可以使用 item as type
const myCanvas = document.getElementById("main_canvas") as HTMLCanvasElement;
//也可以使用&lt;type&gt;item
const myCanvas = &lt;HTMLCanvasElement&gt;document.getElementById("main_canvas");
```

如果在运行时，类型断言错误了，那么什么都不会发生。不会抛出错误也不会以null填充
## 限制
这种断言的类型转换只能转为比本类型更精确的类型，而不能把两个无关的类型互转

如果一定要转的话必须先转为any或者unknown
```ts
const a = (expr as any) as T;
```

## 非null/undefined断言
变量名后面加一个感叹号，表示ts无需检查这个变量是否为null或undefined（仅限strictNullChecks开启时）
```ts
console.log(x!.toFixed());
```

# 字面量类型
定义一个只能是一个量的类型

如果和联合类型一起使用，就能创造出指定某几种输入值的函数（类似于enum）
```ts
function printText(s: string, alignment: "left" | "right" | "center") {
  // ...
}
printText("Hello, world", "left");
printText("G'day, mate", "centre");//报错
```
数字也行：
```ts
function compare(a: string, b: string): -1 | 0 | 1 {
  return a === b ? 0 : a > b ? 1 : -1;
}
```
字面量和非字面量也可以结合起来用
```ts
interface Options {
  width: number;
}
function configure(x: Options | "auto") {
  // ...
}
configure({ width: 100 });
configure("auto");
configure("automatic");//报错
```

## 收字面量的函数不收string怎么办
```ts
declare function handleRequest(url: string, method: "GET" | "POST"): void;

const req = { url: "https://example.com", method: "GET" };
handleRequest(req.url, req.method);//报错，参数类型不对不能输入
```
- 方法一：req.method as "GET"
- 方法二：const req = { url: "https://example.com", method: "GET" } **as const**;
  - **as const 可以将前面的对象类型里面的属性全部变成字面量**，ts认为里面的属性不会再改变，method的类型就是"GET"
