# struct

- 使用`type Vertex struct{...}`定义一个结构体
- 使用`v := Vertex{1, 2}`初始化一个该结构体的值
- 对于指向一个结构体值的指针`p`，可直接读写其属性，会自动解引用`p.X = 1e9`

```go
package main

import "fmt"

type Vertex struct {
	X int
	Y int
}

func main() {
	v := Vertex{1, 2}
	p := &v
	p.X = 1e9
	fmt.Println(v)
}
```

## struct 字面量

```go
var (
	v1 = Vertex{1, 2}  // 有类型 Vertex
	v2 = Vertex{X: 1}  // Y:0 是隐式的
	v3 = Vertex{}      // X:0 且 Y:0
	p  = &Vertex{1, 2} // 有类型 *Vertex
)
```
