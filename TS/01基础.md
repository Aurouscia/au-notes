# 安装
要使用typeScript，需要在命令行输入：

npm install -g typescript

这会让npm添加一个全局的typescript包(在PATH里面)，可以通过命令行使用tsc命令

# 类型指定
使用冒号进行类型指定

a:string = "xxx"<br/>
function funA(a:string):string{}

使用interface定义一个静态的对象结构
<pre>
interface Person {
    firstName:string;
    lastName:string;
}
var p1:Person = {
    firstName:"A",
    lastName:"B"
}
</pre>

# 常用类型
## 原始
- string  字符串
- number  数字，没有int和float，只有number
- boolean 布尔

注意都是小写，区分于String

## 数组
- string\[ \]
- number\[ \]
- boolean\[ \]<br/>
区分于\[number\]
### 同义的另一种写法：
Array&lt;number&gt;

## any
var xxx:any

表示xxx可以是任何类型，让ts放弃对其的检查

# 函数
<pre>
function fun1(string:参数类型):返回值类型
{函数体}
</pre>

## 匿名函数
大多数情况下匿名函数可以自动推断出类型
<pre>
const names = ["Alice", "Bob", "Eve"];
 
// Contextual typing for function - parameter s inferred to have type string
names.forEach(function (s) {
  console.log(s.toUpperCase());
})
</pre>

# 匿名对象类型
pt: { x: number; y: number }
<pre>
function printCoord(pt: { x: number; y: number }) {
  console.log("The coordinate's x value is " + pt.x);
  console.log("The coordinate's y value is " + pt.y);
}
printCoord({ x: 3, y: 7 });
</pre>
## 标注为可选属性
属性名称后加一个问号即可

obj: { first: string; last?: string }

<span style="color:red">可选属性在读取前务必检查是否为undefined</span>
