# type

go 有以下的基本类型：

- bool
- string
- int  int8  int16  int32  int64
- uint uint8 uint16 uint32 uint64 uintptr
- byte // alias for uint8
- rune // alias for int32
       // represents a Unicode code point
- float32 float64
- complex64 complex128

除非有特别的理由，否则应该使用 int 和 uint（根据目标架构是 32 还是 64 位自动选择长度）

# 零值

未初始化的变量会使用零值

- 对于数字类型是 0
- 对于布尔类型是 false
- 对于字符串是 ""

# 类型转换

在 go 中，类型转换必须使用 `T()` 显式转换

```go
i := 42
f := float64(i)
u := uint(f)
```

# 自动数值类型

对于未标注的数值变量初始化，类型会根据数值字面量自动推断：

```go
i := 42           // int
f := 3.142        // float64
g := 0.867 + 0.5i // complex128
```
