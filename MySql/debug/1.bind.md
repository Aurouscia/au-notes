# 无法从本机之外连接
```sh
vim /etc/my.cnf # mysql 配置文件（可能在其他路径）
```

```ini
[mysqld]
bind-address = 0.0.0.0 #（允许所有IP连接）（要配合防火墙使用）
```

```sh
service mysqld restart # 重启 mysql
```

## 诊断无法连接的原因

- 长时间卡住 / Can't connect to server on 'xxx'
    - 表示被防火墙拦截
- Host 'xxx' is not allowed to connect to this MySQL server
    - 表示本设备不在 mysql 的 `bind-address` 中，或用户名的 `host` 不包含本机 IP （参考 user@host.md 文件）
