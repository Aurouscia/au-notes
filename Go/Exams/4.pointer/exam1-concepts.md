# 指针概念题

## 一、填空题

1. 在 Go 中，`&x` 表示获取变量 x 的________。
    - 指针（内存地址）

2. 如果 `p` 是一个指针变量，`*p` 表示________。
    - p 指向的变量值

3. 声明一个指向 `int` 类型的指针变量，应该写为________。
    - var *int

4. 指针的零值是________。
    - nil

5. 函数 `new(int)` 返回的类型是________。
    - *int

## 二、判断题（正确填✓，错误填✗）

1. Go 语言支持指针运算（如 `p++`）。
    - 错

2. 对 nil 指针进行解引用操作会导致编译错误。
    - 错，是运行时 panic

3. 使用指针作为函数参数可以避免大结构体的拷贝，提高效率。
    - 对

4. 方法接收者为指针类型时，方法内部可以修改原对象的字段。
    - 对

5. `*int` 和 `*float64` 是相同的指针类型。
    - 错

## 三、代码阅读题

请写出以下代码的输出结果：

### 题目 1

```go
package main

import "fmt"

func main() {
    x := 10
    p := &x
    *p = 20
    fmt.Println(x)
}
```

输出：________
- 20

### 题目 2

```go
package main

import "fmt"

func modify(y *int) {
    *y = *y + 10
}

func main() {
    x := 5
    modify(&x)
    fmt.Println(x)
}
```

输出：________
- 15

### 题目 3

```go
package main

import "fmt"

func main() {
    var p *int
    if p == nil {
        fmt.Println("nil")
    }
    fmt.Println("end")
}
```

输出：________
- nil

### 题目 4

```go
    p1 = p2

    fmt.Println(b)

```

输出：
- a = ________

    - 100

## 四、简答题

1. 简述在什么情况下应该使用指针作为函数参数？
    - 需要修改原值时
    - 原值是大结构体，需要避免复制以提高效率时

2. 值接收者方法和指针接收者方法有什么区别？什么时候应该选择指针接收者？
    - 值接收者方法只能读取原值字段，不能修改（如果修改，改的也是复制的副本）
    - 指针接收者方法可以修改原值字段

3. 以下代码有什么问题？请说明原因。

```go
func getPointer() *int {
    x := 10
    return &x
}
```
- 返回局部变量的指针（逃逸），不推荐这么做
