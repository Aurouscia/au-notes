# 包（package）

- go 程序由包（package）构成
- go 程序由名为 main 的包开始运行
- 以下程序使用了“导入路径”为 `fmt` 和 `math/rand` 两个包
- 按约定，包的名字与导入路径最后一段一致，例如 `math/rand` 的包名应该是 `rand`

```go
package main

import (
	"fmt"
	"math/rand"
)

func main() {
	fmt.Println("My favorite number is", rand.Intn(10)) // 包名.函数名
}

```

# import

以下两种写法等价，但更推荐第二种“括号写法”（ide 也会自动生成第二种）

```go
import "fmt"
import "math"
```

```go
import (
	"fmt"
	"math"
)
```

# 导出

- 一个包内，大写开头的名称代表它能被外部使用
- 反之，小写的代表私有

```go
func main() {
	fmt.Println(math.Pi)
}
```
