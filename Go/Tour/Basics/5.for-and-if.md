# for

go 仅支持 for 一种循环语句，用法就是经典的 初始化-判断-更新 三板斧（区别是外面不要小括号）

```go
func main() {
	sum := 0
	for i := 0; i < 20; i++ {
		sum += i
	}
	fmt.Println(sum)
}
```

## for 的省略形式

- 初始化和更新部分可以省略（分号也应省略），这种情况下 `for condition` 相当于 js 中的 `while(condition)`
- 条件部分也可以省略，这种情况下 `for` 相当于 js 中的 `while(true)` 死循环

```go
func main() {
	sum := 1
	for ; sum < 1000; {
		sum += sum
	}
	fmt.Println(sum)
}
```

# if

用法和经典的 if-else 语句一样（区别是条件表达式外不要小括号）

## if 初始化

if 语句可以在条件表达式前添加一个初始化语句，用分号分隔，例如 `v := 1; v > 0`，这个 `v` 仅在 if 的表达式和语句块内可用

```go
if v := 1; v > 0 {
	fmt.Println(v)  // v 在这里可用
}
// v 在这里不可用
```

## if 和 for 练习：实现开方函数

```go
package main

import (
	"fmt"
)

func Sqrt(x float64) float64 {
	z := 1.0
	for {
		var diff = (z*z - x) / (2*z)
		if diff < 0.0001 && diff > -0.0001 {
			return z
		}
		z -= diff
	}
}

func main() {
	fmt.Println(Sqrt(2))
}

```

# switch

go 的 switch 和经典 switch 的不同：

- case 条件无需是常量，可以是表达式
- case 命中后仅执行该 case （相当于自动加了break）

```go
func main() {
	fmt.Print("Go runs on ")
	// switch 也可以有初始化表达式，写在分号前半边即可，分号后半边是用来比较的值表达式
	switch os := runtime.GOOS; os {
	case "darwin":
		fmt.Println("macOS.")
	case "linux":
		fmt.Println("Linux.")
	default:
		// freebsd, openbsd,
		// plan9, windows...
		fmt.Printf("%s.\n", os)
	}
}
```

## 省略表达式的 switch

相当于 switch true，可用于替代长长的 if else 链条

```go
switch {
	case t.Hour() < 12: // 条件为真时进入
		fmt.Println("Good morning!")
	case t.Hour() < 17:
		fmt.Println("Good afternoon.")
	default:
		fmt.Println("Good evening.")
}
```
