# 数组与切片概念题

## 一、填空题

1. 在 Go 中，`[3]int` 和 `[5]int` 是________（相同/不同）的类型。
    - 不同

2. 数组是________类型，切片是________类型。
    - 值
    - 引用

3. 切片的三个核心属性是：________、________、________。
    - T 元素类型
    - len 长度
    - cap 容量

4. 使用 `make([]int, 3, 5)` 创建的切片，长度是________，容量是________。
    - 3
    - 5

5. 从数组 `arr` 创建切片 `s := arr[1:4]`，切片包含的元素是原数组索引从________到________（不含）的元素。
    - 1
    - 4

## 二、判断题（正确填✓，错误填✗）

1. 数组的长度可以在运行时动态改变。
    - 错

2. 切片 `s` 的容量可以通过 `cap(s)` 获取。
    - 对

3. `append` 函数会修改原切片的底层数组。
    - 容量足够时会，反之不会

4. 两个切片可以使用 `==` 直接比较是否相等。
    - 错，切片是引用类型，只有数组可以

5. 当切片的 `len` 等于 `cap` 时，再次 `append` 会分配新的底层数组。
    - 对

## 三、代码阅读题

请写出以下代码的输出结果：

### 题目 1

```go
package main

import "fmt"

func main() {
    arr := [3]int{1, 2, 3}
    b := arr
    b[0] = 100
    fmt.Println(arr[0])
    fmt.Println(b[0])
}
```

输出：
- `arr[0]` = ________
    - 1
- `b[0]` = ________ 
    - 100

### 题目 2

```go
package main

import "fmt"

func main() {
    arr := [5]int{10, 20, 30, 40, 50}
    s1 := arr[1:4]
    s2 := append(s1, 99)
    
    fmt.Println(s2)
    fmt.Println(arr)
}
```

输出：
- `s2` = ________
    - [20, 30, 40, 99]
- `arr` = ________
    - [10, 20, 30, 40, 99]

### 题目 3

```go
package main

import "fmt"

func main() {
    arr := [3]int{1, 2, 3}
    s := arr[:]
    s = append(s, 4)
    s[0] = 100
    
    fmt.Println(arr)
    fmt.Println(s)
}
```

输出：
- `arr` = ________
    - [1, 2, 3]
- `s` = ________
    - [100, 2, 3, 4]

## 四、简答题

1. 简述数组和切片的主要区别（至少列出 3 点）。
    - 数组是值类型，切片是引用类型
    - 数组是固定长度的，切片的长度可变
    - 切片较常用，数组使用较少


2. 为什么 `append` 操作需要写成 `s = append(s, x)` 而不是直接 `append(s, x)`？
    - 因为 append 不改变原切片，而是返回一个新切片



3. 从数组创建切片后，修改切片元素可能会影响原数组，请说明什么情况下会影响，什么情况下不会影响。
    - 在容量足够时会继续使用原数组（修改有影响）容量不足后自动更换为容量更大的数组（修改不影响）
