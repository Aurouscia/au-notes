# print

## print命令
zsh内建的print命令，比echo更强大

## print -P
启用提示符扩展，支持颜色代码和特殊序列

```bash
# 基本用法
print -P "%F{red}警告信息%f"

# 在脚本中的使用示例
print -P "%F{red}⚠️  Git user.name 不是 $my_name（当前：${name:-空}）已经重设%f"
print -P "%F{red}⚠️  Git user.email 不是 $my_email（当前：${email:-空}）已经重设%f"
```

## 颜色代码
- `%F{red}` - 设置前景色为红色
- `%f` - 重置前景色

## 参数扩展
- `${name:-空}` - 如果name为空或未设置，使用默认值"空"
