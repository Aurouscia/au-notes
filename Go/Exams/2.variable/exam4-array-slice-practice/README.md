# 数组与切片实践题

## 题目要求

请完成以下 3 个函数的实现，所有代码写在 `main.go` 中。

---

### 任务 1：数组反转

实现函数 `ReverseArray`，接收一个整数数组，返回反转后的新数组（不修改原数组）。

```go
func ReverseArray(arr [5]int) [5]int
```

示例：
- 输入：`[1, 2, 3, 4, 5]`
- 输出：`[5, 4, 3, 2, 1]`

---

### 任务 2：切片去重

实现函数 `Unique`，接收一个整数切片，返回去重后的新切片（保持原有顺序）。

```go
func Unique(s []int) []int
```

示例：
- 输入：`[]int{1, 2, 2, 3, 3, 3, 4}`
- 输出：`[]int{1, 2, 3, 4}`

---

### 任务 3：矩阵转置

实现函数 `Transpose`，接收一个二维切片（矩阵），返回其转置矩阵。

```go
func Transpose(matrix [][]int) [][]int
```

示例：
- 输入：`[[1, 2, 3], [4, 5, 6]]`（2行3列）
- 输出：`[[1, 4], [2, 5], [3, 6]]`（3行2列）

---

## 验证

在 `main` 函数中调用以上函数并打印结果，验证实现是否正确。
