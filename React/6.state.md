# useState
需要使用一个响应式变量时，使用 useState 方法。
```jsx
const [count, setCount] = useState(0);
```
count可以在模板中使用，setCount可以修改count的值（同时立即触发重新渲染）  
约定：这一对值应该命名为`[thing, setThing]`

## hook
这种use开头的函数被称为hook，hook是react提供的一类特殊函数，用于在函数组件中使用状态和其他react特性。  
hook必须在组件的顶层调用，不能在条件语句、循环、嵌套函数中调用。

## 与vue的区别
useState类似于vue中的ref  
state需要通过对应的函数来设置，而vue中需要通过赋值.value来设置

## 快照
```jsx
    <>
      <h1>{number}</h1>
      <button onClick={() => {
        setNumber(number + 1);
        setNumber(number + 1);
        setNumber(number + 1);
      }}>+3</button>
    </>
```
在上述例子中，点击“+3”按钮也只能使number+1  
这是由于setNumber只会设置**下一次**渲染所使用的值  
所以第二、第三次调用setNumber时，number的值并不是+1后的值，而是这次渲染时的原始number值。

## 更新函数
如果在setNumber时需要基于“其他setNumber调用后的值”来设置新值，需要使用更新函数  
向setNumber传入一个函数即可。
```jsx
    <>
      <h1>{number}</h1>
      <button onClick={() => {
        setNumber(n => n + 1);
        setNumber(n => n + 1);
        setNumber(n => n + 1);
      }}>+3</button>
    </>
```
