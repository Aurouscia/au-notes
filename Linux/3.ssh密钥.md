# ssh-key-gen

免密登录 ssh 的最简单方案，且不会影响原登录方式

## 1.生成密钥对

```sh
ssh-keygen -t ed25519 -C "your_email@example.com"
# 按回车，使用默认路径 & 不设密码
```

## 2.公钥发给服务器

```sh
ssh-copy-id user@remote_host
```

如果上述工具没有，可以改为手动操作：

- 确保服务器有 .ssh 目录
    - `mkdir -p ~/.ssh && chmod 700 ~/.ssh`
- 移动公钥
    - `scp .\.ssh\id_xxx.pub user@remote_host:/tmp/id_xxx.pub`
    - windows 的 .ssh 目录在 USERPROFILE 下
- 追加到 authorized_keys 文件
    - `cat /tmp/id_xxx.pub >> ~/.ssh/authorized_keys`
