# 移动端安全区
在一些移动端浏览器中，`100vh`并不是真正的高度，用100vh可能导致画面底部溢出

## 获取高度
使用`env()`取得安全区高度
```css
.body{
    top: env(safe-area-inset-top, 0)
    bottom: env(safe-area-inset-bottom, 0); /* env：取指定名称环境变量，若没有就用第二参数兜底 */
}
```
`env()`的返回值本质上是一个带单位的length，可以作为`calc()`的参数使用

## 不使用env的类似做法
- `window.innerHeight`就是扣除安全区的高度
- `--`开头的值作为css变量，通过js全局设置，css中用`var()`取用
    ```js
    function setRealHeight() {
    // 把 px 值写进 :root
    document.documentElement.style.setProperty('--vh', window.innerHeight * 0.01 + 'px');
    }
    setRealHeight();           // 首次
    window.addEventListener('resize', setRealHeight);   // 旋转/地址栏变化
    window.addEventListener('orientationchange', setRealHeight);
    ```
    ```css
    .footer {
    /* 用 calc 把 100vh 换成真正的 100*var(--vh) */
    bottom: calc(100 * var(--vh, 1vh) - 100%);
    /* 或者更直观：height + translate */
    height: 50px;
    transform: translateY(calc(100 * var(--vh, 1vh) - 100% - 50px));
    }
    ```
