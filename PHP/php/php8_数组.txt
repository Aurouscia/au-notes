经过我的验证，php里数组也是值传递的

数组的下标可以是整数或者字符串
元素顺序不是下标决定的，而是由加入的顺序决定
$arr=array(元素1，元素2，xxx);
$arr=array(1,4,3.14,"abc",true,false)可以混着存储各种类型，默认下标
$arr=array(2=>1,5=>5,7=>'abc')下标可以任意设定，不用顺序，不用连续
如果有的没有设定下标，就会自动分配默认下标，规则是：
前面已经使用过的最大数字+1
浮点数作为下标：直接抹掉小数点后的东西
纯数字字符串：当作数字看待
布尔：true为1，false为0
同名下标：后面的会覆盖前面的
也可以：
$arr1[]=1;
$arr1[]=5;会自动往后添加
$arr['aa']=1;

关联数组：指下标为字符串的数组
索引数组：指下标为0开始的连续数字的数组

数组遍历
【foreach】($arr as [$key =>] $value)可以称$key为键变量，$value为值变量
{这里可以对$key和$value做操作，会从第一个到最后一个按顺序来}
在循环开头先取到值，马上指针向右移动一位
数组有个指针，几个函数依赖于指针：
$v1=current($arr1)取得指针当前的键值
$v1=key($arr1)取得指针当前指着的键名
$v1=next($arr1)将指针移向下一个元素并取值
$v1=prev($arr1)将指针移向上一个元素并取值
$v1=reset($arr1)将指针移向第一个元素并取值
$v1=end($arr1)将指针移向最后一个元素并取值
$v1=each($arr1)取得当前元素的下标和值然后将指针移向下一位置
其得到的是一个有四个元素的数组，值和键名各两份，方便用不同方式读取
array(1=>值，'value'=>值，0=>键名，'key'=>键名)
【通过for和next遍历】
reset($arr1);
$len=count($arr1);
for($i=0;$i<$len;$i++){
	$key=key($arr1);
	$value=current($arr1);
	next($arr1);
}
list()函数
list($v1,$v2,$v3,$v4)=$arr1;
相当于
$v1=$arr1[0];
$v2=$arr1[1];
$v3=$arr1[2];
$v4=$arr1[3];
【while】
reset($arr1)
while(list($key,$value)=each($arr1))最后，each会返回false，让循环结束
{
对$key和$value操作
}

foreach也可以break和continue
传递过来的方式是值传递，可以设置为引用传递
foreach($arr as $key => &$value)
这时操作会改变原数组中的值

如果在foreach过程中对数组进行了增加，删除，某种指针性操作
foreach会不受影响，把原来的数组复制一份继续遍历，新数组的指针会停在原处

冒泡排序法：
从左到右两两比较数字大小，如果左边大于右边就交换位置
第一遍之后最大的数肯定在最右边了
所以第二遍不需要到达最后一位
第三遍不需要到达倒数第二位
一共需要经历n-1次循环
第一次需要比较n-1次

选择排序法
取得数组最大值，并与最后一项交换
第二次，取得数组除最后一项外的最大值，并与倒数第二项一项交换
循环要经历n-1次

查找：找到了返回数组下标，没找到false
结果可以与false严格比较===以免与0混淆

in_array(要判断的元素,数组)
array_merge(数组1，数组2)
array_unique()去掉重复元素
