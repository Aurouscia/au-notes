# 函数

签名和 ts 的函数一样，除了：
- 类型标注前没有冒号
- 关键字是更简短的 `func`

```go
package main

import "fmt"

func add(x int, y int) int {
	return x + y
}

func main() {
	fmt.Println(add(42, 13))
}

```

# 类型标注省略

如果多个参数类型一致，只需要写最后一个

```go
func add(x, y int) int {
	return x + y
}
```

# 多个返回值

函数可以返回多个返回值（没有括号的元组）

```go
package main

import "fmt"

func swap(x, y string) (string, string) {
	return y, x
}

func main() {
	a, b := swap("hello", "world")
	fmt.Println(a, b)
}

```

# 具名返回值

函数结尾可以定义具名返回值（如下），函数内直接给这些定义好的“返回用”变量赋值，最后跟一个裸 return，外部即可接收到

注意：这种写法在较长的函数中破坏可读性

```go
package main

import "fmt"

func split(sum int) (x, y int) {
	x = sum * 4 / 9
	y = sum - x
	return
}

func main() {
	fmt.Println(split(17))
}
```
